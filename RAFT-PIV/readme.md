# 使用RAFT-net进行PIV速度场计算



## 问题概述

基于预训练好的raft-net进行PIV速度场计算，

结果表明，RAFT-net可以有效计算出PIV速度场，展现出极高的精度，其可靠性还有待进一步的探究。

RAFT-net使用内存较大，推理速度较低。
